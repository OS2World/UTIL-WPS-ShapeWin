
treyes                  OS/2 User's Manual                  treyes

NAME

    treyes - Shape Extension 付きの xeyes もどき

SYNOPSIS

    treyes [ -option ...]

DESCRIPTION

    背景透過で、マウスを追いかける目玉を表示します。

    一応、普通のウィンドウと同様に、ウィンドウ上で
    
    ・右ボタン押下      ウィンドウが前面に表示されます
    ・左ボタン押下      メニューが表示されます
    ・左ボタンドラッグ  ウィンドウを移動させることができます

    となっています。
    
        低速マシンですと、ドラッグ開始から、ドラッグフレームが表示される
        までに結構時間がかかります。
    
    メニューからは

        ┌──┐
	│Move│        ウィンドウを移動します
	├──┤
	│Size│        ウィンドウのサイズを変更します
	├──┤
	│Hide│        ウィンドウを隠します
	├──┤
	│Exit│        プログラムを終了します
	└──┘

    が選択できます。サイズ変更では、メニュー選択後、マウスを動かした側の
    辺、あるいはコーナにポインタが移動し、その方向のリサイズが可能になり
    ます。

OPTIONS

    -x #        水平位置を指定します
    -y #        垂直位置を指定します
    -w #        幅を指定します
    -h #        高さを指定します

    オプションによる指示が無かった場合には、
    
        位置        マウスの位置
	サイズ      スクリーンサイズの１／８

    になります。

使用上の注意

    低速のマシンで、あまり大きなサイズにすると、ＷＰＳ／ＰＭの動作が極端
    に遅くなります。どの程度の負荷になるか試してから大きくしていくように
    してください。
